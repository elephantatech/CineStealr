# CineStealr

Local AI application for generating cinematic scene descriptions from images using LLaVA and YOLO.

## Getting Started

1. **Prerequisites:**
   - Podman & Podman Compose
   - Python 3.9+
   - Node.js & Bun

2. **Start Services:**
   ```bash
   podman compose up -d
   ```

3. **Access App:**
   - Frontend: http://localhost:5173
   - Backend API: http://localhost:8000
